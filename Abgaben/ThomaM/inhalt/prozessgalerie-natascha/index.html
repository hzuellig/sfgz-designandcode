Natascha

Wichtig bei der Prozessgalerie war mir, nicht nur die verschiedenen Design abzubilden,
sondern auch der Code. Denn für mich gehört der Code genau so zum Designprozess wie das Endresultat.
Zudem habe ich den Fokus darauf gelegt, nicht mehrere Versionen von einer Programmierung zu zeigen,
sondern eher den Fokus auf die verschiedensten Möglichkeiten von Visualisierungen zu lenken.
